# Ereninki's PC Status Selfbot

This selfbot is changing your status by the usage of your pc

# Tech Stacks

- Typescript
- Slack Bolt
- [systeminformation](https://www.npmjs.com/package/systeminformation)

# Dependeinces

- bun
- git

  ***

# Installation

you just need to add a **.env** file and follow the **.env.example** as base. You can just edit the values of the **.env.example** file and remove the **.example** from the file and use it. Run the **unix.sh** for mac os and linux or **windows.bat** for windows to install the dependiencies and main code and you start the selfbot by running the **start.bat** for windows and **start.sh** for mac os and linux!!
